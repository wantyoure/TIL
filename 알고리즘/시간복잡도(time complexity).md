시간복잡도

함수를 실행 시간이 어느정도일지
표현해보고싶다

실행시간(running time)이란 :
함수/알고리즘 수행에 필요한 
스텝(step) 수 

각 라인을 수행하기 위해 필요한 스텝 수는 상수(constant)라고 가정하자

(표로 다시 재배치)
cost        times
c1           1
c2          n+1
c3            n
c4            1

N -> 커질수록
-N이 커질수록 덜 중요한 것은 제거
-최고차항만 의미있다
-최고차항의 계수는 의미없다
